## Overview
This project is a prototype smart parking system for the City of Moondalup. The system tracks available parking bays in real-time, displays weather and community messages, and stores scanned license plates of vehicles. It is designed to work in uncontrolled carparks and is scalable for future enhancements.

The project demonstrates key competencies in **object-oriented programming**, **file operations**, **unit testing**, and **version control**.

---

## Features
- Tracks real-time availability of carpark bays.
- Displays ambient temperature and announcements.
- Stores scanned license plates.
- Robust and scalable design for future updates.
- Configurable carpark setup via direct call or configuration file.
- Demonstrates core OOP concepts and best coding practices.
